# DRM3 Homebrew Tap

Homebrew formulae for DRM3 software on macOS.

## Install

```bash
brew tap drm3labs/drm3
brew install mor
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `mor` | Local AI gateway for Morpheus decentralized inference |

## Other Platforms

Linux, Ubuntu, and Chromebook — see [drm3-releases](https://github.com/drm3labs/drm3-releases).
